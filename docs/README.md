# Demo-GIF vervangen door een echte schermopname

`scriba-demo.gif` is een **illustratieve mockup** in de Scriba-huisstijl. Wil je
de echte app laten zien? Maak een korte schermopname en vervang het bestand.

## Zo maak je een goede demo-GIF (Windows)

1. Installeer **[ScreenTo​Gif](https://www.screentogif.com/)** (gratis, open source).
2. Start Scriba en zet het venster op een nette grootte (±720×460).
3. Klik in ScreenToGif op **Recorder**, sleep het kader over het Scriba-venster.
4. Neem een korte flow op (10–15 s): opname starten → praten → stoppen →
   transcript verschijnt → eventueel ✨ Vat samen.
5. Klik **Stop**, knip overbodige frames weg, en exporteer als **GIF**.
6. Sla op als `docs/scriba-demo.gif` (zelfde naam) en commit. De README toont
   hem dan automatisch.

Tip: houd de GIF onder ~5 MB (kortere opname, ~10 fps) zodat hij snel laadt op
de GitHub-pagina.
